Recruiter-Job Seeker Bridge

Recruiter-Job Seeker Bridge (Jan, 2024): Developed a web application designed to streamline the hiring process for recruiters and job seekers. The project effectively bridges the gap between both parties, facilitating easier and more efficient selection and recruitment. Implemented features to enhance user experience, ensuring seamless interaction and communication. This solution significantly improved the hiring workflow, making it more accessible and user-friendly for all stakeholders. Tech Stack: HTML, CSS, Bootstrap, JavaScript, DBMS.

Prediction Model

Prediction Model (September, 2024): Developed a car price prediction model using supervised learning techniques. Leveraged the Car Price Prediction Dataset to analyze key attributes such as mileage, age, and brand. Employed Mean Absolute Error (MAE) as the loss function to evaluate model performance. Implemented Mini-Batch Gradient Descent for optimization to enhance model accuracy. Successfully predicted car prices with significant precision, providing valuable insights for potential buyers and sellers.
